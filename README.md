## 🚀 Platform Features

This project includes a custom-built **Lead Management & Automation System**:

- **Automated Booking Flow:** Seamless client intake with instant validation.
- **Bi-Directional Sync:** Leads are stored in a local dashboard AND mirrored to **Google Sheets** in real-time.
- **Intelligent Notifications:** 
  - **Admin Alert:** Instant email notification for every new booking.
  - **Client Confirmation:** Professional automated response with booking details and a 48h turnaround promise.
- **Sequential ID System:** Professional lead tracking using unique identifiers (e.g., `BX01`, `BX02`).
- **Secure Admin Dashboard:** JWT-protected portal for managing leads, adding team members, and exporting data to CSV.
